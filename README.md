

# Spam Detection System 

This project focuses on building an intelligent Spam Detection System that classifies messages (SMS/Email) as Spam or Ham (Not Spam) using Machine Learning techniques.

🚀 Features

Text preprocessing (removing stopwords, punctuation, etc.)
Feature extraction using TF-IDF / Count Vectorizer
Model training using algorithms like:
Naive Bayes
Logistic Regression
Support Vector Machine (SVM)
High accuracy spam classification
Easy-to-use and scalable pipeline

🧠 Technologies Used

Python
Scikit-learn
Pandas & NumPy
Natural Language Processing (NLP)

📂 Dataset

Public spam datasets (e.g., SMS Spam Collection Dataset)
Contains labeled messages: spam or ham

📊 Workflow

Data Collection
Data Cleaning & Preprocessing
Text Vectorization
Model Training
Model Evaluation
Prediction on new messages

📈 Results
Achieved high accuracy and precision in detecting spam messages
Efficient in minimizing false positives.
